# 开发

## 说明

按照`VuePress`官网介绍，请`NPM`安装依赖会生成错误的依赖树，建议采用`YARN`安装；本文档用`NPM`示例启动及构建。

## 克隆仓库
```shell
git clone https://github.com/beautifulBoys/xinUI-vue.git
```

## 安装依赖
```shell
yarn
```

## 组件开发
```shell
npm run dev
```

## 组件构建
```shell
npm run build
```

## 文档站开发
```shell
npm run dev-docs
```

## 文档站构建
```shell
npm run build-docs
```
