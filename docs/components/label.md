# Label 表单标签

## 引用
```js
import { Label } from 'xin-ui'
```

## 示例
<example-label/>

## 参数

### width

* 类型：`String`
* 默认值：`""`
* 描述：表单标签宽度。

### align

* 类型：`String`
* 默认值：`"left"`
* 描述：对齐方式，`["left", "center", "right"]`。

### required

* 类型：`Boolean`
* 默认值：`false`
* 描述：是否必须，表单中使用。

## 事件

无事件。
