# Loading 加载条

## 引用
```js
import { Loading } from 'xin-ui'
```

## 示例
<example-loading/>

## 参数

### status

* 类型：`String`
* 默认值：`""`
* 描述：加载条状态。

## 事件

### start
* 描述：开始加载。
* 回调参数: 无。

### success
* 描述：加载成功。
* 回调参数: 无。

### error
* 描述：加载失败。
* 回调参数: 无。

## 依赖情况

无依赖。
