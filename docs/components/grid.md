# Grid 栅格布局

## 引用
```js
import { Grid } from 'xin-ui'
```

## 示例
<example-grid/>

## 参数

### gutter

* 类型：`Number`
* 默认值：`0`
* 描述：间距。

### align

* 类型：`String`
* 默认值：`"top"`
* 描述：水平分布模式，同`align-items`属性。

### justify

* 类型：`String`
* 默认值：`"center"`
* 描述：垂直分布模式，同`justify-content`属性。

### className

* 类型：`String`
* 默认值：`"xin-row-content"`
* 描述：该属性暂未实现。
