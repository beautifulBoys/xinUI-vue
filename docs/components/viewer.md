# Viewer 预览器

## 说明

目前仅支持预览图片，文件类型尚未支持。

## 引用
```js
import { Viewer } from 'xin-ui'
```

## 示例
<div>
  <example-viewer/>
</div>

## 参数

### list

* 类型：`Array`
* 默认值：`[]`
* 描述：数据源。

### index

* 类型：`Number`
* 默认值：`0`
* 描述：默认展示的图片序号。

## 事件

无事件。

## 依赖情况

依赖`Icon`组件。
