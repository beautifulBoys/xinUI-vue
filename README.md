# xinUI-vue

## 介绍

xinUI组件库开发历时半年，由开发者@beautifulBoys开发及维护；目前开发已完前两个阶段，日趋完善。

* 独立自主开发，不拿来主义。
* 与业务完全分离，专注于UI设计及交互逻辑部分。
* 所有组件样式、色彩搭配均为自主设计。
* 以组件模块为开发基本单位。
* 每个组件都可单独引入使用；也可整体引用。
